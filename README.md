# Sistema de Hospital

Um sistema de gerenciamento hospitalar desenvolvido em **C**, utilizando **ponteiros** e conceitos fundamentais de programação estruturada.  
O projeto foi criado com fins acadêmicos e de aprendizado, servindo como base para explorar estruturas de dados, manipulação de memória e boas práticas de organização de código.

---

## Funcionalidades

- Cadastro de pacientes (nome, idade, CPF, etc.)
- Cadastro de médicos (nome, especialidade, CRM, etc.)
- Registro de consultas
- Listagem de pacientes e médicos
- Busca por CPF ou CRM
- Exclusão de registros
- Uso de **ponteiros** para manipulação dinâmica de dados

---

## Tecnologias Utilizadas

- Linguagem C
- Estruturas de dados (structs)
- Ponteiros
- Manipulação de arquivos (se aplicável)
- Compilador GCC ou similar
